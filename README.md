# 123Pan CV

适用于 **123云盘** 的提取码助手脚本。  
支持自动保存提取码、文件名，表格查看，点击复制，导入导出，支持 **Tampermonkey 云同步**。  

---

## ✨ 功能
- ✅ 自动保存文件名 + 提取码  
- ✅ 表格化查看，文件名可点击跳转到真实分享链接  
- ✅ 提取码点击即可复制  
- ✅ 支持导出 / 导入 / 清空  
- ✅ 支持 **Tampermonkey 自带云同步**  

---

## 🛠 使用方法
1. 打开 **123云盘** 提取页面，脚本会自动捕获文件名 + 提取码  
2. 右键 Tampermonkey 图标 → 找到脚本菜单：  
   - 📑 查看提取码记录  
   - 💾 导出提取码  
   - 📥 导入提取码  
   - 🗑️ 清空所有记录  
3. 表格里：  
   - 点击 **文件名** → 跳转到真实分享链接  
   - 点击 **提取码** → 一键复制
